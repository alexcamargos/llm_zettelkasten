Este arquivo é a **síntese viva** do cofre Zettelkasten em alto nível. Não substitui o `index.md` nem as notas de detalhe; resume o que a base cobre hoje, os riscos epistémicos dominantes e as prioridades de exploração quando isso for inferível a partir do índice e das notas.

O conteúdo abaixo será **regenerado** pelo agente durante o fluxo `/lint`, após auditoria de ligações, contradições e lacunas. Até a primeira execução de `/lint` com cofre populado, este texto permanece genérico. Quando existirem notas de literatura e permanentes, a síntese deve refletir domínios cobertos, fragilidades (por exemplo dependência de fontes web de **confidence** baixa) e ausências óbvias de ligação no grafo.

Use o `index.md` para navegação linha a linha e o `/recall` para perguntas pontuais. Volte a executar `/lint` periodicamente para manter este panorama alinhado com a evolução real do conhecimento armazenado.
